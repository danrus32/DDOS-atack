To install and run, we write these commands in the Linux terminal \ in the thermux:

                                                    apt upgrade
                                                    apt update 
                                                    apt install git
                                                    apt install python 
                                                    git clone https://github.com/danrus32/DOS-atack
                                                    cd DOS-atack
                                                    chmod 777 dos.py
                                                    ./dos.py
